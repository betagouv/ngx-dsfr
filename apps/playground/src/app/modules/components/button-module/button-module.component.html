DsfrButtonComponent

The DsfrButtonComponent purpose is to represent the button

It is based on the Button open_in_new component in the DSFR open_in_new .

Use the --dsfr-button-width CSS variable to customize the button's width.

It has several @Input() and @Output() properties:

It is required when no icon is provided. It will display an icon, alone if no label provided, or at the left or right according to the iconAlignment Input.
Here open_in_new is the list of all the icons available. It will align the provided icon at the left or right of the label. This input is not necessary if no label is provided. For accessibility, it will display the title on hover of the button. For accessibility, the aria-controls attribute identifies the element whose contents or presence are controlled by the element on which the attribute is set. It emits the Event value on click on the button.
