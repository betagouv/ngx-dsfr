DsfrErrorPageComponent

The DsfrErrorPageComponent purpose is to represent an error page in an application.

It is based on the Error Page open_in_new component in the DSFR open_in_new .

According to the status code you provide, default templates, with default texts will be displayed, following the recommendations of the DSFR documentation.

You can also provide your own template, with your own text, via Content Projection open_in_new .

It has several @Input() properties:

It will be displayed in a H1 tag The status code for the error we want to display Link to the homepage of your application Link to the contact page of your application.
