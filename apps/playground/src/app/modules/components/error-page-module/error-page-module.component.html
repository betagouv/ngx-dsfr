DsfrErrorPageComponent

The DsfrErrorPageComponent purpose is to represent an error page in an application.

It is based on the Error Page open_in_new component in the DSFR open_in_new .

It has several @Input() properties:

The name of your web application.
When a 503 error status happens, the name is displayed on the default text. It will be displayed on a H1 tag. Link to the homepage of your application Link to the contact page of your application.
