DsfrNavigationComponent

The DsfrNavigationComponent purpose is to represent the navigation of an application.

It is based on the Navigation open_in_new component in the DSFR open_in_new .

It has only one @Input() property:

The value must satisfy the Navigation interface : 
  interface NavigationItem extends LinkNavigation {
      child?: ChildNavigation;
  }
  interface LinkNavigation {
      id: string;
      label: string;
      href: string;
      children?: LinkNavigation[];
      userRoles?: UserRole[];
  }
  interface ChildNavigation {
      isMega?: boolean;
      title?: string;
      description?: string;
      children: LinkNavigation[];
  }
  export type Navigation = NavigationItem[];
