DsfrPasswordComponent

The DsfrPasswordComponent purpose is to represent the password input form field

It is based on the Input Password open_in_new component in the DSFR open_in_new .

It has several @Input() properties:

It will display a description below the label It will display a placeholder for the corresponding input tag It will display the component with the error state : label, input and conditons will be in red. It will display the component with the success state : label, input and conditons will be in green. It will display a link below the input, when the Component has type='login', which redirect to the forgotten password dedicated page. If nothing is provided, the link will not be displayed You can see the recommendations concerning how to secure a password open_in_new .
The value must satisfy the PasswordParams interface : an object with the minSize, minSpecialCharacters and minDigitalCharacters properties, with number values

Password field for sign up It displays all the params to define and check a strong password

Example : 

        <dsfr-password
          id="2"
          type="signup"
          placeholder="Enter your password..."
          label="This is a label"
          hint="This is a hint"
        ></dsfr-password>
Password field for login It displays only the form field and, optionally, the "Forgot Password" link

Example : 

        <dsfr-password
          id="2"
          type="login"
          placeholder="Enter your password..."
          label="This is a label"
          hint="This is a hint"
          forgotPasswordLink="/modules/password"
        ></dsfr-password>
