DsfrSelectComponent

The DsfrSelectComponent purpose is to represent a select form field.

It is based on the Liste dÃ©roulante open_in_new component in the DSFR open_in_new .

It has several @Input() properties:

The label of the select component The ID used for a specific instance of the DsfrSelectComponent. The value must be an array of objects satisfying the SelectOption interface : each object must have an id property, a label property and a value property. All these properties have string values. They represent the different options we can select from the Component. An optional text that is displayed in the Component by default, when no option is selected yet. Whether we can select multiple options or not. An optional text to help end-users understand the purpose of a select. It is a way to give more details and complement the label. Whether a select is disabled or not. Whether the control is in error mode or not. The error message to be displayed when in error mode. Whether the control is in success mode or not. The success message to be displayed when in success mode. Whether to show the selected values as deletable tags under the select or not.
