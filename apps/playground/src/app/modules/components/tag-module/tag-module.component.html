DsfrTagComponent

The DsfrTagComponent purpose is to represent a tag element. Use it to categorize / classify / organize content via keywords.

It is based on the Tag open_in_new component in the DSFR open_in_new .

It has several @Input() properties and two @Output() properties:

The text displayed inside the tag. It is mandatory. You can import the TagType enum, from the @betagouv/ngx-dsfr/tag, in your app, to safely provide a value for this property. An optional icon for the tag. It's not taken into account if the tag is deletable or selectable and can only be placed at the left of the text. You can import the ElementSize enum, from the @betagouv/ngx-dsfr library entry-point, in your app, to safely provide a value for this property. The link to which a clickable tag redirects once it's clicked upon. It only works for, and is MANDATORY for, clickable tags. Whether or not the tag is selected initially. It's only taken into account if the tag is selectable. Emits a boolean to indicate if the tag has been selected or deselected. It's only taken into account if the tag is selectable. Emits an event when the tag is deleted. It's only taken into account if the tag is deletable.
