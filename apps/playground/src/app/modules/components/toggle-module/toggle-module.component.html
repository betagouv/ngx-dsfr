DsfrToggleComponent

The DsfrToggleComponent purpose is to represent the toggle element.

It is based on the Toggle open_in_new component in the DSFR open_in_new .

It has several @Input() properties and one @Output() property:

It is required and given to the id attribute of the underlying input tag. It is also used to generate the aria-describedby attribute The label of the toggle when it's checked. The label of the toggle when it's not checked. The label of the toggle (description). It will display a description below the label The value for the disabled attribute, for the underlying input tag Emits the value of the toggle as boolean when the user toggles it
