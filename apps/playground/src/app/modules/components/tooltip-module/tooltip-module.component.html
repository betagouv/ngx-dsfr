DsfrTooltipComponent

The DsfrTooltipComponent purpose is to represent a tooltip element.

It is based on the Tag open_in_new component in the DSFR open_in_new .

It has several @Input() properties:

The text displayed inside the tooltip. It is mandatory. The text displayed in the link that triggers the tooltip. It is mandatory. The id of the tooltip. It is mandatory. You can import the TooltipTriggerType enum, from the @betagouv/ngx-dsfr/tooltip, in your app, to safely provide a value for this property.
