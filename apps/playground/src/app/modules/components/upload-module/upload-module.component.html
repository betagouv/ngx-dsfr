DsfrUploadComponent

The DsfrUploadComponent purpose is to represent an input file form field

It is based on the Upload open_in_new component in the DSFR open_in_new .

It has several @Input() and Output() properties:

It is required and given to the id attribute of the underlying input tag. It is also used to generate the aria-describedby attribute and IDs for the error message if needed. It is used for the name attribute of the underlying input tag The label that will be displayed on top of the form field The descriptive text that will be displayed above the label. It must contain all the expected formats/weights/sizes It will allow the fact that the user can upload more than only 1 file It will display the component with the error state : label, input and the then required error message will be in red It will display the failure message below the input tag in red when the property hasFailed is true. The value for the disabled attribute, for the underlying input tag Emits the value of the input on change event.
